## 一、产品愿景

**一句话**：像素游戏资产一站式生成平台，从2D图片到3D场景预览全覆盖。

**目标**：让没有像素艺术功底的开发者也能快速生成游戏级Sprite和动画。

## 四、Phase 3 详细规格（3D预览）

### 核心场景
```
┌─────────────────────────────────────────┐
│              3D 预览画布                 │
│                                         │
│     ┌───────────────────┐              │
│     │   Billboard Sprite│              │
│     │   (像素Sprite)     │              │
│     └───────────────────┘              │
│                                         │
│  [正面] [侧面] [背面] [顶视] [自由]      │
│                                         │
│  光照：[环境光] ████░░ [平行光] ███░░░  │
└─────────────────────────────────────────┘
```

### 技术方案
- **渲染引擎**：Three.js
- **Sprite类型**：THREE.Sprite + THREE.Billboard
- **相机类型**：
  - PerspectiveCamera（自由视角）
  - OrthographicCamera（正交视角，预设角度）
- **光照**：AmbientLight + DirectionalLight

### 功能点拆解
| 功能 | 描述 | 技术点 |
|------|------|--------|
| 场景搭建 | 3D场景初始化 | Scene/Camera/Renderer |
| Sprite显示 | 像素图作为Billboard | THREE.Sprite + Texture |
| 自由视角 | 轨道控制器旋转 | OrbitControls |
| 预设视角 | 正交相机快速切换 | setView(front/side/back/top) |
| 光照调节 | 双光源强度控制 | light.intensity |
| 背景切换 | 场景背景设置 | scene.background |

### 交互设计
1. 用户点击"3D预览"Tab
2. Sprite自动放置在场景中心
3. 默认显示正面视角（正交）
4. 底部工具栏：视角切换 / 光照调节
5. 右侧属性面板：Sprite缩放 / 旋转
