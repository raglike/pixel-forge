---

## 一、项目概述

**一句话描述**：将图片转换为像素风格动画，支持预览和导出，兼容主流游戏引擎。

**核心价值**：
- 降低像素游戏制作门槛
- 一键生成游戏精灵（Sprite）
- 直接导出引擎可用格式

---

## 二、用户场景

| 场景 | 描述 |
|------|------|
| 独立游戏开发者 | 快速生成像素角色/场景动画 |
| 像素艺术爱好者 | 将照片/图片像素化创作 |
| 游戏工作室 | 批量制作游戏资产 |

---

## 三、核心功能需求

### 3.1 图片转像素动画
- 上传图片（支持PNG/JPG/WebP）
- 设置目标分辨率（8x8 到 256x256）
- 设置调色板（可选：自动提取/预设/自定义）
- 生成像素化结果

### 3.2 动画制作
- 多帧动画支持（2-16帧）
- 帧时间间隔设置（ms）
- 关键帧编辑（可选）
- 洋葱皮功能（可选）

### 3.3 预览
- 实时动画预览
- 播放/暂停/逐帧控制
- 缩放查看

### 3.4 导出格式
| 格式 | 用途 |
|------|------|
| GIF | 预览/分享 |
| 序列帧（PNG/JPG） | 通用素材 |
| Sprite Sheet | Unity/Godot 直接导入 |
| Aseprite (.ase) | Aseprite 编辑器 |
| JSON metadata | 引擎解析用（帧数/尺寸/间隔） |
