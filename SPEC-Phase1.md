## 一、产品定位

**一句话**：将图片一键转换为像素风格动画，导出游戏引擎可用格式。

**差异化**：竞品（Piskel/Aseprite）无AI辅助；我们主打"上传即用，无需设计功底"

**目标用户**：
- 独立游戏开发者（快速生成Sprite）
- 像素艺术爱好者（照片像素化创作）
- 游戏工作室（批量资产制作）

---

## 二、功能规格

### 2.1 P0 核心功能（14天MVP）

#### F1: 图片上传与像素化
- **输入**：PNG/JPG/WebP/GIF
- **分辨率选项**：8×8, 16×16, 32×32, 64×64, 128×128, 256×256 + 自定义
- **算法**：最近邻插值（保持硬边缘）
- **预览**：实时像素化效果
- **交互**：拖拽上传 + 点击选择

#### F2: 调色板处理
- **自动模式**：中位切分算法（Median Cut）
- **预设调色板**：PICO-8(16色)、NES(54色)、GameBoy(4色灰度)、灰度、256色
- **颜色数量**：2-256色滑块调节
- **主色调检测**：color-thief提取图片主导颜色

#### F3: 帧管理
- **多帧导入**：GIF分解为帧 / 批量上传序列帧
- **时间轴UI**：缩略图网格，支持拖拽排序
- **帧操作**：添加空白帧、删除、复制
- **帧间隔**：10ms-2000ms可配置

#### F4: 动画导出
| 格式 | 用途 | 配置项 |
|------|------|--------|
| GIF | 预览/分享 | 帧率(8/12/15/24/30fps)、循环次数 |
| PNG序列帧 | 通用素材 | ZIP打包下载 |
| Sprite Sheet + JSON | Unity/Godot | 行列布局、metadata格式 |

---

### 2.2 P1 增强功能（Phase 2）

- **洋葱皮**：显示前/后帧半透明叠加，调节透明度
- **动画预设**：走路循环(4/6/8帧)、待机、攻击序列
- **导出Aseprite格式**：.ase + .json

---

### 2.3 P2 高级功能（Phase 3+）

- **AI补间动画**：自动生成中间帧
- **AI风格迁移**：照片→像素风格智能转换
- **颜色推荐**：基于图片内容推荐调色板

## 三、技术架构

### 3.1 技术栈

```
前端框架: React 18 + TypeScript
样式: Tailwind CSS
图像处理: Canvas 2D API (原生)
GIF编码: gif.js (Web Worker加速)
GIF解码: omggif (导入现有GIF)
调色板: color-thief + 自研MedianCut
状态管理: Zustand
打包: Vite
部署: Vercel / 静态托管
```
