# PixelForge 产品白皮书

> **版本**：v1.0  
> **日期**：2026-04-22  
> **产品定位**：像素游戏创作工作台 + PV宣发工具

---

## 一、产品愿景

**一句话**：让像素创作者从"做出作品"到"展示作品"零距离。

PixelForge解决了像素创作的两个断点：
1. **创作断点**：从照片/图片到像素游戏资产的专业流水线
2. **宣发断点**：从静态素材到动态PV展示的一键转化

---

## 二、核心产品矩阵

### 2.1 PixelForge 像素创作工作台

**定位**：专业级像素游戏资产生成工具

**核心功能**：
| 功能 | 说明 |
|------|------|
| AI像素化 | 照片/图片→像素艺术（多种算法） |
| xBR放大 | 无损像素放大2x/3x/4x |
| 调色板库 | 50+精选调色板（游戏主机/像素艺术/复古/现代） |
| 动画模板 | 20+动画模板（Idle/Walk/Run/Attack/Death等） |
| 动画状态机 | 可视化状态编辑+帧序列绑定 |
| 帧事件触发器 | 在帧上标记事件点，导出JSON供游戏引擎 |
| 蓝图导出 | 可视化蓝图PNG，帧序列+事件+时长一览 |
| Sprite Sheet导出 | 批量导出PNG+JSON，兼容主流游戏引擎 |

**目标用户**：
- 独立游戏开发者
- 像素艺术爱好者
- 游戏美术学生
- 像素内容创作者

**在线地址**：https://raglike.github.io/pixel-forge/

## 三、场景闭环分析

### 3.1 闭环一：像素游戏从0到宣发

**场景描述**：创作者有一张照片，想做成像素游戏并发布宣发素材。

**传统路径**（耗时3-7天）：
```
照片 → Photoshop像素化 → 调色 → 动画帧制作 → 导出Sprite → Unity/Godot导入 → 编写动画逻辑 → PR制作软件 → 导出PV
```

**PixelForge路径**（耗时1-2小时）：
```
照片 → PixelForge像素化 + 调色板 + 动画模板 → 直接导出Sprite Sheet → Unity/Godot导入
        ↓
    Pixel PV制作游戏PV → 发布
```

**价值提升**：
- 时间：3-7天 → 1-2小时
- 成本：PR软件订阅/外包PV → 零成本
- 门槛：需要PR技能 → 会打字就会做PV

**典型用户故事**：
> "我用PixelForge把自拍做成了像素角色，花了10分钟。又花了5分钟在Pixel PV里选了个'战场'模板，输入角色名'勇者'，直接导出了一个战斗PV。第二天我把游戏demo发到itch.io，PV直接用上了。"——独立游戏开发者小B

## 四、技术架构

### 4.1 串联方案

**方案A：数据对接**
```
PixelForge → 导出Sprite Sheet + 动画数据JSON → Pixel PV读取 → 作为PV素材
```

**方案B：URL参数传递**
```
PixelForge "制作PV"按钮 → 打开Pixel PV + URL参数（调色板/模板/动画数据）
```

**方案C：独立工作流**
```
PixelForge做素材 → 用户手动导入Pixel PV → 选择模板 → 生成PV
```

**推荐方案B**：最小开发成本，用户体验最流畅

### 4.2 URL参数设计

```typescript
interface PVParams {
  template?: 'battle' | 'cyber' | 'adventure' | 'minimal';
  text?: string;
  fontSize?: number;
  color?: string;
  strokeColor?: string;
  strokeWidth?: number;
  spriteUrl?: string;  // 可选：Sprite Sheet URL
  palette?: string[];  // 可选：调色板
}

// 示例
https://raglike.github.io/pixel-pv/?template=battle&text=勇者&color=%23ff4444
```

### 4.3 数据流

```
用户上传照片
    ↓
PixelForge处理
    ↓
┌───────────────────────────────────────┐
│  素材输出                              │
├───────────────────────────────────────┤
│  PNG Sprite Sheet（角色/场景）          │
│  JSON动画数据（帧数/时长/事件）          │
│  PNG蓝图预览                           │
│  JSON调色板                            │
└───────────────────────────────────────┘
    ↓                    ↓
 导入游戏引擎      URL参数跳转Pixel PV
                       ↓
              ┌──────────────────────┐
              │   Pixel PV处理       │
              ├──────────────────────┤
              │  应用调色板           │
              │  叠加视觉特效         │
              │  添加文字动画         │
              │  渲染导出PV          │
              └──────────────────────┘
                       ↓
                  导出PNG/视频
                       ↓
                  社交媒体/游戏平台发布
```
