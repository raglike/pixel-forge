## 六、功能细化清单（Phase 2）

### F10: 洋葱皮功能
```
需求描述:
  - 在当前帧上叠加显示前/后帧的半透明图像
  - 帮助动画师对齐动作，保持运动连续性

功能点:
  [ ] 洋葱皮开关按钮
  [ ] 透明度滑块 (0-100%, 默认30%)
  [ ] 显示前帧/后帧选项
  [ ] 快捷键: O 切换开关
  [ ] 快捷键: [ / ] 切换前/后帧
  [ ] 多帧叠加 (可选显示2-3帧)

技术方案:
  - 在Canvas上叠加绘制前/后帧
  - 使用 globalAlpha 控制透明度
  - 使用 CompositeOperation: 'source-over'

依赖: 帧管理 (Phase 1)
优先级: ★★★
```

### F11: GIF智能分解
```
需求描述:
  - 自动分析GIF的帧数和时序
  - 智能检测并合并重复帧
  - 批量导入时自动处理

功能点:
  [ ] GIF文件拖入自动分解
  [ ] 显示原始帧数和处理后帧数
  [ ] 重复帧检测开关
  [ ] 相同帧合并预览
  [ ] 手动调整分解结果

技术方案:
  - 使用 omggif 库解析GIF
  - 比较帧数据检测重复
  - 保留第一帧，后续重复帧引用

依赖: 图片上传, 帧管理
优先级: ★★
```

### F12: 动画预设模板
```
需求描述:
  - 提供常用的动画模板
  - 用户快速套用生成循环动画
  - 支持自定义预设保存

功能点:
  [ ] 预设列表:
      - 走路循环 (4帧/6帧/8帧)
      - 待机/Idle (2帧/4帧)
      - 攻击 (3帧/5帧/6帧)
      - 跳跃 (4帧)
      - 死亡 (4帧/6帧)
  [ ] 帧间距设置
  [ ] 模板预览动画
  [ ] 自定义预设保存
  [ ] 预设导出/导入

技术方案:
  - 预设数据: { name, frames, frameDelay, loop }
  - 本地存储: localStorage
  - JSON格式导入导出

依赖: 帧管理
优先级: ★★★
```

### F13: 自定义调色板编辑器
```
需求描述:
  - 用户可创建和管理自定义调色板
  - 从图片中吸取颜色
  - 支持导入导出标准格式

功能点:
  [ ] 调色板列表 (创建/删除/重命名)
  [ ] 颜色块操作:
      - 添加颜色 (颜色选择器)
      - 删除颜色 (右键/双击)
      - 拖拽排序
  [ ] 吸管工具 (从图片取色)
  [ ] 调色板设置:
      - 颜色数量限制
      - 排序方式 (色相/亮度/饱和度)
  [ ] 导入格式: .pal, .hex, .json
  [ ] 导出格式: .pal, .hex, .json, PNG色板图

技术方案:
  - 颜色选择器: React Colorful
  - 吸管: Canvas getImageData + 颜色量化
  - 存储: localStorage + JSON

依赖: 调色板系统 (Phase 1)
优先级: ★★★
```

### F14: Aseprite格式导出
```
需求描述:
  - 导出Aseprite编辑器兼容格式
  - 包含完整的图层和帧信息

功能点:
  [ ] 导出 .ase 文件
  [ ] 导出 .json 元数据 (Aseprite格式)
  [ ] 配置:
      - 图层命名
      - 帧标签
      - 标签颜色
  [ ] 预览导出内容

技术方案:
  - .ase格式: 二进制文件格式 (参考Aseprite文档)
  - .json: Aseprite JSON导出格式兼容

依赖: 调色板系统, 帧管理
优先级: ★★
```
